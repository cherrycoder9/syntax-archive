# 자바 언어 기초

## 연산자

### 비교 연산자

#### 기본 비교 연산자

자바에서 두 값을 비교하는 데 사용되는 연산자들임.

```java
int a = 10, b = 20;

boolean isEqual = (a == b);         // 같음: false
boolean isNotEqual = (a != b);      // 다름: true
boolean isGreater = (a > b);        // 크다: false
boolean isLess = (a < b);           // 작다: true
boolean isGreaterOrEqual = (a >= b);// 크거나 같다: false
boolean isLessOrEqual = (a <= b);   // 작거나 같다: true

System.out.println("a == b: " + isEqual);
System.out.println("a != b: " + isNotEqual);
System.out.println("a > b: " + isGreater);
System.out.println("a < b: " + isLess);
System.out.println("a >= b: " + isGreaterOrEqual);
System.out.println("a <= b: " + isLessOrEqual);
```

#### 객체 비교

참조 타입의 비교는 주소 값을 비교함. 내용 비교를 위해서는 `equals()` 메소드를 사용해야 함.

```java
String str1 = "Hello";
String str2 = "Hello";
String str3 = new String("Hello");

boolean refComparison = (str1 == str2);        // true (같은 문자열 리터럴)
boolean contentComparison1 = str1.equals(str2);// true
boolean contentComparison2 = str1.equals(str3);// true

System.out.println("str1 == str2: " + refComparison);
System.out.println("str1.equals(str2): " + contentComparison1);
System.out.println("str1.equals(str3): " + contentComparison2);
```

#### null 비교

객체가 null인지 확인할 때는 `==` 연산자를 사용함.

```java
String str = null;
boolean isNull = (str == null);  // true
boolean isNotNull = (str != null);  // false

System.out.println("str is null: " + isNull);
System.out.println("str is not null: " + isNotNull);
```

#### 부동소수점 비교

부동소수점 숫자의 비교는 정확하지 않을 수 있으므로 주의가 필요함.

```java
double d1 = 0.1 + 0.2;
double d2 = 0.3;

System.out.println("d1 == d2: " + (d1 == d2));  // false

// 권장되는 방법: 허용 오차 사용
final double EPSILON = 0.00001;
boolean isAlmostEqual = Math.abs(d1 - d2) < EPSILON;
System.out.println("d1 ≈ d2: " + isAlmostEqual);  // true
```

#### 문자 비교

문자(char)는 내부적으로 정수 값으로 처리되므로 비교 연산자를 직접 사용할 수 있음.

```java
char ch1 = 'A';
char ch2 = 'B';

System.out.println("ch1 < ch2: " + (ch1 < ch2));  // true
System.out.println("'a' > 'A': " + ('a' > 'A'));  // true (소문자가 더 큰 ASCII 값을 가짐)
```

#### instanceof 연산자

객체가 특정 클래스의 인스턴스인지 확인하는 데 사용됨.

```java
String str = "Hello";
Integer num = 10;

System.out.println("str instanceof String: " + (str instanceof String));  // true
System.out.println("num instanceof Integer: " + (num instanceof Integer));  // true
System.out.println("str instanceof Object: " + (str instanceof Object));  // true (모든 객체는 Object의 인스턴스)
```

#### 비교 연산자와 조건문

비교 연산자는 주로 조건문과 함께 사용됨.

```java
int age = 18;
boolean isAdult = age >= 18;

if (isAdult) {
    System.out.println("성인입니다.");
} else {
    System.out.println("미성년자입니다.");
}

// 삼항 연산자와 함께 사용
String status = (age >= 18) ? "성인" : "미성년자";
System.out.println("상태: " + status);
```

#### 예제: 다양한 비교 연산 활용

```java
public class ComparisonOperatorsDemo {
    public static void main(String[] args) {
        // 기본 타입 비교
        int x = 5, y = 10;
        System.out.println("x < y: " + (x < y));
        System.out.println("x > y: " + (x > y));
        System.out.println("x == y: " + (x == y));

        // 문자열 비교
        String str1 = "Hello";
        String str2 = new String("Hello");
        System.out.println("str1 == str2: " + (str1 == str2));
        System.out.println("str1.equals(str2): " + str1.equals(str2));

        // 부동소수점 비교
        double d1 = 0.1 + 0.2;
        double d2 = 0.3;
        System.out.println("d1 == d2: " + (d1 == d2));
        System.out.println("Math.abs(d1 - d2) < 0.00001: " + (Math.abs(d1 - d2) < 0.00001));

        // null 비교
        String nullStr = null;
        String notNullStr = "Not Null";
        System.out.println("nullStr == null: " + (nullStr == null));
        System.out.println("notNullStr != null: " + (notNullStr != null));

        // instanceof 사용
        System.out.println("str1 instanceof String: " + (str1 instanceof String));
        System.out.println("nullStr instanceof String: " + (nullStr instanceof String));

        // 조건문에서의 사용
        int age = 20;
        if (age >= 18) {
            System.out.println("성인입니다.");
        } else {
            System.out.println("미성년자입니다.");
        }

        // 문자 비교
        char ch1 = 'A', ch2 = 'a';
        System.out.println("'A' < 'a': " + (ch1 < ch2));
    }
}
```
