# 자바 언어 기초

## 연산자

### 비트 연산자

#### 기본 비트 연산자

자바에서 비트 단위의 연산을 수행하는 연산자들임.

```java
int a = 5;  // 0101 in binary
int b = 3;  // 0011 in binary

int andResult = a & b;   // 비트 AND: 0001 (1 in decimal)
int orResult = a | b;    // 비트 OR: 0111 (7 in decimal)
int xorResult = a ^ b;   // 비트 XOR: 0110 (6 in decimal)
int notResult = ~a;      // 비트 NOT: 11111111111111111111111111111010 (-6 in decimal, due to two's complement)

System.out.println("a & b = " + andResult);
System.out.println("a | b = " + orResult);
System.out.println("a ^ b = " + xorResult);
System.out.println("~a = " + notResult);
```

#### 시프트 연산자

비트를 왼쪽 또는 오른쪽으로 이동시키는 연산자임.

```java
int num = 8;  // 1000 in binary

int leftShift = num << 2;   // 왼쪽으로 2비트 시프트: 100000 (32 in decimal)
int rightShift = num >> 1;  // 오른쪽으로 1비트 시프트: 0100 (4 in decimal)
int unsignedRightShift = -8 >>> 1;  // 부호 없는 오른쪽 시프트

System.out.println("8 << 2 = " + leftShift);
System.out.println("8 >> 1 = " + rightShift);
System.out.println("-8 >>> 1 = " + unsignedRightShift);
```

#### 비트 마스킹

특정 비트만 조작하거나 추출하는 기법임.

```java
int flags = 0b1010;  // 10 in decimal
int mask = 0b1100;   // 12 in decimal

int result = flags & mask;  // 1000 in binary (8 in decimal)
System.out.println("Masked result: " + result);

// 특정 비트 설정
int setBit = flags | 0b0001;  // 1011 in binary (11 in decimal)
System.out.println("Set bit result: " + setBit);

// 특정 비트 클리어
int clearBit = flags & ~0b0010;  // 1000 in binary (8 in decimal)
System.out.println("Clear bit result: " + clearBit);
```

#### 비트 플래그

여러 불리언 값을 하나의 정수로 표현할 수 있음.

```java
final int READ = 1;     // 0001
final int WRITE = 2;    // 0010
final int EXECUTE = 4;  // 0100

int permissions = READ | WRITE;  // 0011

boolean canRead = (permissions & READ) != 0;
boolean canWrite = (permissions & WRITE) != 0;
boolean canExecute = (permissions & EXECUTE) != 0;

System.out.println("Can read: " + canRead);
System.out.println("Can write: " + canWrite);
System.out.println("Can execute: " + canExecute);
```

#### 2의 거듭제곱 계산

시프트 연산자를 사용하여 2의 거듭제곱을 빠르게 계산할 수 있음.

```java
for (int i = 0; i < 10; i++) {
    int powerOfTwo = 1 << i;
    System.out.println("2^" + i + " = " + powerOfTwo);
}
```

#### 비트 연산을 이용한 짝수/홀수 판별

마지막 비트를 확인하여 짝수/홀수를 판별할 수 있음.

```java
for (int i = 0; i < 5; i++) {
    String parity = (i & 1) == 0 ? "짝수" : "홀수";
    System.out.println(i + "는 " + parity + "입니다.");
}
```

#### 예제: 비트 연산자의 다양한 활용

```java
public class BitwiseOperationsDemo {
    public static void main(String[] args) {
        // 기본 비트 연산
        int a = 60;  // 0011 1100
        int b = 13;  // 0000 1101
        System.out.println("a & b = " + (a & b));   // 12 (0000 1100)
        System.out.println("a | b = " + (a | b));   // 61 (0011 1101)
        System.out.println("a ^ b = " + (a ^ b));   // 49 (0011 0001)
        System.out.println("~a = " + (~a));         // -61 (1100 0011)

        // 시프트 연산
        int num = 8;
        System.out.println("8 << 2 = " + (num << 2));  // 32
        System.out.println("8 >> 2 = " + (num >> 2));  // 2
        System.out.println("-8 >>> 2 = " + (-8 >>> 2));  // 1073741822

        // 비트 마스킹
        int flags = 0b1010;
        System.out.println("Is third bit set? " + ((flags & 0b0100) != 0));  // false

        // 비트 플래그
        final int READ = 1, WRITE = 2, EXECUTE = 4;
        int permissions = READ | WRITE;
        System.out.println("Can execute? " + ((permissions & EXECUTE) != 0));  // false

        // 2의 거듭제곱 계산
        System.out.println("16 is 2^" + Integer.numberOfTrailingZeros(16));

        // 짝수/홀수 판별
        System.out.println("15 is " + (15 % 2 == 0 ? "even" : "odd"));
        System.out.println("15 is " + ((15 & 1) == 0 ? "even" : "odd"));

        // 비트 스왑
        int x = 5, y = 10;
        x = x ^ y;
        y = y ^ x;
        x = x ^ y;
        System.out.println("After swap: x = " + x + ", y = " + y);

        // 부호 변경
        int positive = 5;
        int negative = ~positive + 1;
        System.out.println("Negative of 5: " + negative);

        // 최소값과 최대값 찾기
        int min = Math.min(x, y);
        int max = Math.max(x, y);
        System.out.println("Min: " + min + ", Max: " + max);
    }
}
```
