# 자바 언어 기초

## 제어문

### break와 continue

#### break 문

반복문이나 switch 문을 즉시 종료시키는 데 사용됨.

```java
for (int i = 0; i < 10; i++) {
    if (i == 5) {
        break; // i가 5일 때 반복문 종료
    }
    System.out.println("Current number: " + i);
}
System.out.println("Loop ended");
```

#### continue 문

현재 반복을 건너뛰고 다음 반복으로 진행하는 데 사용됨.

```java
for (int i = 0; i < 10; i++) {
    if (i % 2 == 0) {
        continue; // 짝수일 경우 다음 반복으로 건너뜀
    }
    System.out.println("Odd number: " + i);
}
```

#### 중첩 루프에서의 break와 continue

레이블을 사용하여 특정 루프를 제어할 수 있음.

```java
outerLoop:
for (int i = 0; i < 3; i++) {
    for (int j = 0; j < 3; j++) {
        if (i == 1 && j == 1) {
            break outerLoop; // 외부 루프까지 종료
        }
        System.out.println("i: " + i + ", j: " + j);
    }
}
```

#### while 루프에서의 break와 continue

while 루프에서도 break와 continue를 사용할 수 있음.

```java
int i = 0;
while (i < 10) {
    i++;
    if (i % 2 == 0) {
        continue; // 짝수일 경우 다음 반복으로
    }
    if (i > 5) {
        break; // i가 5를 초과하면 루프 종료
    }
    System.out.println("Number: " + i);
}
```

#### do-while 루프에서의 break와 continue

do-while 루프에서도 break와 continue를 사용할 수 있음.

```java
int i = 0;
do {
    i++;
    if (i % 2 == 0) {
        continue; // 짝수일 경우 다음 반복으로
    }
    System.out.println("Odd number: " + i);
    if (i >= 5) {
        break; // i가 5 이상이면 루프 종료
    }
} while (i < 10);
```

#### switch 문에서의 break

switch 문에서 break는 각 case의 실행을 종료하는 데 사용됨.

```java
int day = 3;
switch (day) {
    case 1:
        System.out.println("Monday");
        break;
    case 2:
        System.out.println("Tuesday");
        break;
    case 3:
        System.out.println("Wednesday");
        break;
    default:
        System.out.println("Other day");
}
```

#### 예제: break와 continue의 다양한 활용

```java
public class BreakContinueDemo {
    public static void main(String[] args) {
        // break in for loop
        System.out.println("Break in for loop:");
        for (int i = 0; i < 10; i++) {
            if (i == 5) break;
            System.out.print(i + " ");
        }
        System.out.println("\n");

        // continue in for loop
        System.out.println("Continue in for loop:");
        for (int i = 0; i < 10; i++) {
            if (i % 2 == 0) continue;
            System.out.print(i + " ");
        }
        System.out.println("\n");

        // break in while loop
        System.out.println("Break in while loop:");
        int j = 0;
        while (true) {
            System.out.print(j + " ");
            j++;
            if (j == 5) break;
        }
        System.out.println("\n");

        // continue in do-while loop
        System.out.println("Continue in do-while loop:");
        int k = 0;
        do {
            k++;
            if (k % 2 == 0) continue;
            System.out.print(k + " ");
        } while (k < 10);
        System.out.println("\n");

        // Labeled break
        System.out.println("Labeled break:");
        outerLoop:
        for (int i = 0; i < 3; i++) {
            for (int m = 0; m < 3; m++) {
                if (i == 1 && m == 1) {
                    break outerLoop;
                }
                System.out.print("(" + i + "," + m + ") ");
            }
        }
        System.out.println("\n");

        // break in switch
        System.out.println("Break in switch:");
        int choice = 2;
        switch (choice) {
            case 1:
                System.out.println("Option 1");
                break;
            case 2:
                System.out.println("Option 2");
                break;
            case 3:
                System.out.println("Option 3");
                break;
            default:
                System.out.println("Invalid option");
        }
    }
}
```
