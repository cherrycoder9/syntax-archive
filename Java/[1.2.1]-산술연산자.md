# 자바 언어 기초

## 연산자

### 산술 연산자

#### 기본 산술 연산자

자바에서 기본적인 수학 연산을 수행하는 연산자들임.

```java
int a = 10, b = 3;

int sum = a + b;        // 더하기: 13
int difference = a - b; // 빼기: 7
int product = a * b;    // 곱하기: 30
int quotient = a / b;   // 나누기: 3 (정수 나눗셈)
int remainder = a % b;  // 나머지: 1

System.out.println("10 / 3 = " + (10.0 / 3.0)); // 3.3333333333333335 (실수 나눗셈)
```

#### 증감 연산자

변수의 값을 1 증가시키거나 감소시키는 연산자임.

```java
int count = 0;
count++;  // 후위 증가: 현재 값 사용 후 증가
++count;  // 전위 증가: 증가 후 값 사용

int x = 5, y;
y = x++;  // y는 5, x는 6
y = ++x;  // y는 7, x는 7

System.out.println("x = " + x + ", y = " + y);
```

#### 복합 대입 연산자

연산과 대입을 한 번에 수행하는 연산자임.

```java
int value = 10;
value += 5;  // value = value + 5;와 동일
value -= 3;  // value = value - 3;와 동일
value *= 2;  // value = value * 2;와 동일
value /= 4;  // value = value / 4;와 동일
value %= 3;  // value = value % 3;와 동일

System.out.println("최종 value 값: " + value);
```

#### 문자열 연결 연산자

`+` 연산자는 문자열 연결에도 사용됨.

```java
String str1 = "Hello";
String str2 = "World";
String result = str1 + " " + str2;  // "Hello World"

System.out.println(result);
System.out.println("The answer is " + 42);  // 숫자가 자동으로 문자열로 변환됨
```

#### 연산자 우선순위

수학적 관례를 따르며, 괄호를 사용해 우선순위를 명시적으로 지정할 수 있음.

```java
int result1 = 5 + 3 * 2;       // 11 (곱셈이 먼저)
int result2 = (5 + 3) * 2;     // 16 (괄호 안이 먼저)
int result3 = 10 - 5 + 3;      // 8 (왼쪽에서 오른쪽으로)

System.out.println("Result 1: " + result1);
System.out.println("Result 2: " + result2);
System.out.println("Result 3: " + result3);
```

#### 정수 오버플로우

정수 연산 시 결과가 해당 타입의 범위를 벗어나면 오버플로우가 발생함.

```java
int maxInt = Integer.MAX_VALUE;
System.out.println("Max int: " + maxInt);
System.out.println("Max int + 1: " + (maxInt + 1));  // 오버플로우 발생

long longResult = (long)maxInt + 1;  // long으로 형변환하여 오버플로우 방지
System.out.println("Long result: " + longResult);
```

#### 부동소수점 연산의 주의사항

부동소수점 연산은 정확하지 않을 수 있으므로 주의가 필요함.

```java
double d1 = 0.1;
double d2 = 0.2;
System.out.println("0.1 + 0.2 = " + (d1 + d2));  // 0.30000000000000004

// 금융 계산 등에는 BigDecimal 클래스 사용 권장
import java.math.BigDecimal;

BigDecimal bd1 = new BigDecimal("0.1");
BigDecimal bd2 = new BigDecimal("0.2");
System.out.println("BigDecimal: 0.1 + 0.2 = " + bd1.add(bd2));
```

#### 나눗셈과 0

0으로 나누는 경우, 정수는 예외가 발생하고 부동소수점은 특수한 값을 반환함.

```java
try {
    int result = 10 / 0;  // ArithmeticException 발생
} catch (ArithmeticException e) {
    System.out.println("정수를 0으로 나눌 수 없습니다.");
}

double infinityResult = 10.0 / 0.0;
System.out.println("10.0 / 0.0 = " + infinityResult);  // Infinity

double nanResult = 0.0 / 0.0;
System.out.println("0.0 / 0.0 = " + nanResult);  // NaN (Not a Number)
```

#### 예제: 다양한 산술 연산 활용

```java
public class ArithmeticOperationsDemo {
    public static void main(String[] args) {
        // 기본 산술 연산
        int a = 15, b = 4;
        System.out.println("a + b = " + (a + b));
        System.out.println("a - b = " + (a - b));
        System.out.println("a * b = " + (a * b));
        System.out.println("a / b = " + (a / b));
        System.out.println("a % b = " + (a % b));

        // 증감 연산자
        int count = 0;
        System.out.println("count++: " + count++);
        System.out.println("현재 count: " + count);
        System.out.println("++count: " + ++count);

        // 복합 대입 연산자
        int value = 10;
        value += 5;
        System.out.println("value += 5: " + value);

        // 문자열 연결
        String str = "Java" + " " + "Programming";
        System.out.println(str);

        // 연산자 우선순위
        int result = 10 + 3 * 2;
        System.out.println("10 + 3 * 2 = " + result);

        // 정수 오버플로우
        int maxValue = Integer.MAX_VALUE;
        System.out.println("Max value: " + maxValue);
        System.out.println("Max value + 1: " + (maxValue + 1));

        // 부동소수점 연산
        double d1 = 0.1, d2 = 0.2;
        System.out.println("0.1 + 0.2 = " + (d1 + d2));

        // 나눗셈과 0
        try {
            System.out.println("10 / 0 = " + (10 / 0));
        } catch (ArithmeticException e) {
            System.out.println("0으로 나눌 수 없습니다.");
        }
        System.out.println("10.0 / 0.0 = " + (10.0 / 0.0));
    }
}
```
