# 자바 언어 기초

## 제어문

### 조건문 (if, else, switch)

#### if 문

기본적인 조건 분기를 위해 사용됨.

```java
int temperature = 25;

if (temperature > 30) {
    System.out.println("It's hot!");
} else if (temperature > 20) {
    System.out.println("It's warm.");
} else {
    System.out.println("It's cool.");
}
```

#### 삼항 연산자

간단한 if-else 구문을 한 줄로 표현할 수 있음.

```java
int x = 10;
String result = (x > 5) ? "Greater than 5" : "Less than or equal to 5";
System.out.println(result);
```

#### switch 문

여러 가지 경우에 대해 분기할 때 사용됨.

```java
int dayOfWeek = 3;
String day;

switch (dayOfWeek) {
    case 1:
        day = "Monday";
        break;
    case 2:
        day = "Tuesday";
        break;
    case 3:
        day = "Wednesday";
        break;
    case 4:
        day = "Thursday";
        break;
    case 5:
        day = "Friday";
        break;
    case 6:
    case 7:
        day = "Weekend";
        break;
    default:
        day = "Invalid day";
}

System.out.println("Today is " + day);
```

#### Java 12 이상의 switch 표현식

Java 12부터는 switch를 표현식으로 사용할 수 있으며, 화살표 구문과 yield 키워드를 사용할 수 있음.

```java
int dayOfWeek = 3;
String day = switch (dayOfWeek) {
    case 1 -> "Monday";
    case 2 -> "Tuesday";
    case 3 -> "Wednesday";
    case 4 -> "Thursday";
    case 5 -> "Friday";
    case 6, 7 -> "Weekend";
    default -> {
        System.out.println("Invalid day");
        yield "Unknown";
    }
};

System.out.println("Today is " + day);
```

#### 중첩 조건문

조건문 안에 다른 조건문을 넣을 수 있지만, 과도한 중첩은 피해야 함.

```java
int age = 25;
boolean hasLicense = true;

if (age >= 18) {
    if (hasLicense) {
        System.out.println("You can drive.");
    } else {
        System.out.println("You need a license to drive.");
    }
} else {
    System.out.println("You're too young to drive.");
}
```

#### 논리 연산자를 활용한 복합 조건

여러 조건을 조합하여 복잡한 조건식을 만들 수 있음.

```java
int age = 25;
boolean hasLicense = true;
boolean hasInsurance = true;

if (age >= 18 && hasLicense && hasInsurance) {
    System.out.println("You can drive legally.");
} else {
    System.out.println("You cannot drive legally.");
}
```

#### null 체크와 조건문

객체가 null인지 확인할 때 사용됨.

```java
String name = null;

if (name != null && name.length() > 5) {
    System.out.println("Name is longer than 5 characters.");
} else {
    System.out.println("Name is null or not longer than 5 characters.");
}
```

#### 예제: 다양한 조건문 활용

```java
public class ControlFlowDemo {
    public static void main(String[] args) {
        // 기본 if-else
        int score = 85;
        if (score >= 90) {
            System.out.println("Excellent!");
        } else if (score >= 80) {
            System.out.println("Good job!");
        } else if (score >= 70) {
            System.out.println("You passed.");
        } else {
            System.out.println("You need to study more.");
        }

        // 삼항 연산자
        boolean isRaining = true;
        String weather = isRaining ? "Bring an umbrella" : "Enjoy the sun";
        System.out.println(weather);

        // switch 문 (Java 12 이전)
        char grade = 'B';
        switch (grade) {
            case 'A':
                System.out.println("Excellent grade");
                break;
            case 'B':
            case 'C':
                System.out.println("Good grade");
                break;
            case 'D':
                System.out.println("You passed");
                break;
            case 'F':
                System.out.println("Unfortunately you failed");
                break;
            default:
                System.out.println("Invalid grade");
        }

        // switch 표현식 (Java 12 이상)
        int dayNumber = 3;
        String day = switch (dayNumber) {
            case 1, 2, 3, 4, 5 -> "Weekday";
            case 6, 7 -> "Weekend";
            default -> "Invalid day number";
        };
        System.out.println(day);

        // 복합 조건
        int age = 25;
        boolean hasID = true;
        double income = 30000.0;

        if (age >= 21 && hasID && income > 25000) {
            System.out.println("You are eligible for a credit card.");
        } else {
            System.out.println("You are not eligible for a credit card.");
        }

        // null 체크와 문자열 비교
        String input = "Hello";
        if (input != null && input.equalsIgnoreCase("hello")) {
            System.out.println("Greeting received!");
        } else {
            System.out.println("No greeting or invalid input.");
        }
    }
}
```
