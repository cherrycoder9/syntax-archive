# 자바 언어 기초

## 메서드

### 메서드 오버로딩

#### 개요

메서드 오버로딩은 같은 이름의 메서드를 여러 개 정의하는 기법으로, 매개변수의 타입이나 개수가 다르면 가능함.

```java
public class Calculator {
    // 정수 덧셈
    public int add(int a, int b) {
        return a + b;
    }

    // 실수 덧셈
    public double add(double a, double b) {
        return a + b;
    }

    // 세 개의 정수 덧셈
    public int add(int a, int b, int c) {
        return a + b + c;
    }
}
```

#### 오버로딩의 규칙

1. 메서드 이름이 같아야 함
2. 매개변수의 타입이나 개수가 달라야 함
3. 반환 타입만 다른 것은 오버로딩이 아님

#### 오버로딩과 타입 프로모션

자바는 오버로딩된 메서드 중 가장 적합한 메서드를 찾을 때 타입 프로모션을 고려함.

```java
public class TypePromotion {
    public void display(int num) {
        System.out.println("Integer: " + num);
    }

    public void display(double num) {
        System.out.println("Double: " + num);
    }

    public static void main(String[] args) {
        TypePromotion obj = new TypePromotion();
        obj.display(5);     // Integer: 5 출력
        obj.display(5.5);   // Double: 5.5 출력
        obj.display('A');   // Integer: 65 출력 (char가 int로 프로모션)
    }
}
```

#### 가변 인자와 오버로딩

가변 인자를 사용한 메서드도 오버로딩할 수 있지만, 모호성에 주의해야 함.

```java
public class VarargsOverloading {
    public void printNumbers(int... numbers) {
        System.out.println("Varargs method");
        for (int num : numbers) {
            System.out.print(num + " ");
        }
        System.out.println();
    }

    public void printNumbers(int num1, int num2) {
        System.out.println("Two parameters method");
        System.out.println(num1 + " " + num2);
    }

    public static void main(String[] args) {
        VarargsOverloading obj = new VarargsOverloading();
        obj.printNumbers(1, 2);       // Two parameters method 호출
        obj.printNumbers(1, 2, 3, 4); // Varargs method 호출
    }
}
```

### 재귀 메서드

재귀 메서드는 자기 자신을 호출하는 메서드로, 복잡한 문제를 간단한 문제로 나누어 해결할 때 유용함.

```java
public class Recursion {
    public int factorial(int n) {
        if (n <= 1) return 1;
        return n * factorial(n - 1);
    }

    public static void main(String[] args) {
        Recursion obj = new Recursion();
        System.out.println("Factorial of 5: " + obj.factorial(5)); // 120 출력
    }
}
```

#### 재귀의 기본 구조

1. 기본 케이스 (종료 조건)
2. 재귀 케이스 (자기 자신 호출)

#### 다양한 재귀 예제

- 피보나치 수열

```java
public int fibonacci(int n) {
    if (n <= 1) return n;
    return fibonacci(n - 1) + fibonacci(n - 2);
}
```

- 이진 검색

```java
public int binarySearch(int[] arr, int low, int high, int key) {
    if (high >= low) {
        int mid = low + (high - low) / 2;
        if (arr[mid] == key)
            return mid;
        if (arr[mid] > key)
            return binarySearch(arr, low, mid - 1, key);
        return binarySearch(arr, mid + 1, high, key);
    }
    return -1;
}
```

- 하노이 탑

```java
public void hanoi(int n, char from, char to, char aux) {
    if (n == 1) {
        System.out.println("Move disk 1 from " + from + " to " + to);
        return;
    }
    hanoi(n - 1, from, aux, to);
    System.out.println("Move disk " + n + " from " + from + " to " + to);
    hanoi(n - 1, aux, to, from);
}
```

#### 재귀와 반복의 비교

재귀는 코드를 간결하게 만들 수 있지만, 때로는 반복문이 더 효율적일 수 있음.

```java
// 재귀를 사용한 합계 계산
public int sumRecursive(int n) {
    if (n <= 1) return n;
    return n + sumRecursive(n - 1);
}

// 반복문을 사용한 합계 계산
public int sumIterative(int n) {
    int sum = 0;
    for (int i = 1; i <= n; i++) {
        sum += i;
    }
    return sum;
}
```

#### 예제: 오버로딩과 재귀의 조합

```java
public class MethodDemo {
    // 오버로딩된 메서드들
    public int calculate(int a, int b) {
        return a + b;
    }

    public double calculate(double a, double b) {
        return a * b;
    }

    public int calculate(int a, int b, int c) {
        return a + b + c;
    }

    // 재귀 메서드
    public int sumToN(int n) {
        if (n <= 1) return n;
        return n + sumToN(n - 1);
    }

    // 오버로딩된 재귀 메서드
    public int power(int base, int exponent) {
        if (exponent == 0) return 1;
        return base * power(base, exponent - 1);
    }

    public double power(double base, int exponent) {
        if (exponent == 0) return 1;
        return base * power(base, exponent - 1);
    }

    public static void main(String[] args) {
        MethodDemo demo = new MethodDemo();

        // 오버로딩된 메서드 호출
        System.out.println("Sum: " + demo.calculate(5, 3));
        System.out.println("Product: " + demo.calculate(2.5, 4.0));
        System.out.println("Sum of three: " + demo.calculate(1, 2, 3));

        // 재귀 메서드 호출
        System.out.println("Sum to 5: " + demo.sumToN(5));

        // 오버로딩된 재귀 메서드 호출
        System.out.println("2^3: " + demo.power(2, 3));
        System.out.println("2.5^2: " + demo.power(2.5, 2));
    }
}
```
