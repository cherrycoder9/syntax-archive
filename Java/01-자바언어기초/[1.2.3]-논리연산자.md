# 자바 언어 기초

## 연산자

### 논리 연산자

#### 기본 논리 연산자

자바에서 논리적 조건을 조합하거나 부정하는 데 사용되는 연산자들임.

```java
boolean a = true, b = false;

boolean andResult = a && b;  // 논리 AND: false
boolean orResult = a || b;   // 논리 OR: true
boolean notResult = !a;      // 논리 NOT: false

System.out.println("a && b: " + andResult);
System.out.println("a || b: " + orResult);
System.out.println("!a: " + notResult);
```

#### 단락 평가 (Short-circuit Evaluation)

`&&`와 `||` 연산자는 단락 평가를 수행함. 첫 번째 피연산자만으로 결과가 확정되면 두 번째 피연산자는 평가하지 않음.

```java
int x = 5;
boolean result1 = (x > 10) && (x++ < 20);  // x++는 실행되지 않음
System.out.println("result1: " + result1 + ", x: " + x);  // false, 5

boolean result2 = (x < 10) || (x++ > 20);  // x++는 실행되지 않음
System.out.println("result2: " + result2 + ", x: " + x);  // true, 5
```

#### 복합 논리 조건

여러 조건을 조합하여 복잡한 논리식을 만들 수 있음.

```java
int age = 25;
boolean hasLicense = true;
boolean isInsured = true;

boolean canDrive = age >= 18 && hasLicense && isInsured;
System.out.println("Can drive: " + canDrive);  // true

// 괄호를 사용하여 우선순위 지정
boolean complexCondition = (age > 18 || hasLicense) && isInsured;
System.out.println("Complex condition: " + complexCondition);  // true
```

#### 비트 연산자와의 차이

`&`, `|`, `^`는 비트 연산자지만, 불리언 값에 사용하면 논리 연산자처럼 동작함. 단, 단락 평가를 하지 않음.

```java
boolean bitAnd = true & false;  // false
boolean bitOr = true | false;   // true
boolean bitXor = true ^ false;  // true

System.out.println("true & false: " + bitAnd);
System.out.println("true | false: " + bitOr);
System.out.println("true ^ false: " + bitXor);
```

#### 조건부 연산자 (삼항 연산자)

논리 조건에 따라 값을 선택하는 데 사용됨.

```java
int score = 75;
String result = (score >= 60) ? "Pass" : "Fail";
System.out.println("Result: " + result);  // Pass

// 중첩 사용
String grade = (score >= 90) ? "A" : 
               (score >= 80) ? "B" : 
               (score >= 70) ? "C" : 
               (score >= 60) ? "D" : "F";
System.out.println("Grade: " + grade);  // C
```

#### 논리 연산자와 null 체크

객체의 null 체크와 메소드 호출을 조합할 때 유용함.

```java
String str = null;

// NullPointerException 방지
if (str != null && str.length() > 5) {
    System.out.println("String is longer than 5 characters");
} else {
    System.out.println("String is null or not longer than 5 characters");
}
```

#### 예제: 다양한 논리 연산 활용

```java
public class LogicalOperatorsDemo {
    public static void main(String[] args) {
        int x = 5, y = 10;
        boolean condition1 = x < y;
        boolean condition2 = x % 2 == 0;

        // 기본 논리 연산
        System.out.println("x < y AND x is even: " + (condition1 && condition2));
        System.out.println("x < y OR x is even: " + (condition1 || condition2));
        System.out.println("NOT (x < y): " + (!condition1));

        // 단락 평가
        int z = 0;
        boolean shortCircuit = (y > x) || (++z > 0);
        System.out.println("Short-circuit result: " + shortCircuit + ", z: " + z);

        // 복합 조건
        int age = 20;
        boolean hasID = true;
        boolean canBuyAlcohol = age >= 21 && hasID;
        System.out.println("Can buy alcohol: " + canBuyAlcohol);

        // 비트 연산자와 논리 연산자 비교
        boolean b1 = true, b2 = false;
        System.out.println("b1 && b2: " + (b1 && b2));
        System.out.println("b1 & b2: " + (b1 & b2));

        // 삼항 연산자
        int score = 85;
        String gradeMessage = (score >= 90) ? "Excellent" : 
                              (score >= 80) ? "Good" : 
                              (score >= 70) ? "Average" : "Need improvement";
        System.out.println("Grade message: " + gradeMessage);

        // null 체크와 논리 연산
        String str = "Hello";
        if (str != null && !str.isEmpty()) {
            System.out.println("String is not null and not empty");
        }

        // 복잡한 논리 조건
        boolean isWeekend = true;
        boolean isHoliday = false;
        boolean isMorning = true;
        boolean canSleep = (isWeekend || isHoliday) && !isMorning;
        System.out.println("Can sleep in: " + canSleep);
    }
}
```
