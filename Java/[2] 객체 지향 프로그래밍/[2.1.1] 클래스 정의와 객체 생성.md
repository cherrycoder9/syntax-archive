# 객체 지향 프로그래밍

## 클래스와 객체

### 클래스 정의와 객체 생성

#### 클래스 정의

클래스는 객체를 생성하기 위한 템플릿임. 속성(필드)과 행동(메서드)을 포함하며, 객체의 구조와 기능을 정의함. 다음은 `Person` 클래스의 예시임:

```java
public class Person {
    // 필드 정의
    private String name;
    private int age;

    // 생성자
    public Person(String name, int age) {
        this.name = name;
        this.age = age;
    }

    // 메서드
    public void introduce() {
        System.out.println("안녕하세요, 제 이름은 " + name + "이고 " + age + "살입니다.");
    }

    // Getter와 Setter
    public String getName() { return name; }
    public void setName(String name) { this.name = name; }
    public int getAge() { return age; }
    public void setAge(int age) { this.age = age; }
}
```

#### 객체 생성

클래스를 정의한 후, `new` 키워드를 사용해 객체를 생성함. 이를 인스턴스화라고 부름:

```java
// Person 객체 생성
Person person1 = new Person("김철수", 25);
Person person2 = new Person("이영희", 30);

// 메서드 호출
person1.introduce(); // 출력: 안녕하세요, 제 이름은 김철수이고 25살입니다.
person2.introduce(); // 출력: 안녕하세요, 제 이름은 이영희이고 30살입니다.
```

#### 필드와 메서드 접근

객체의 필드와 메서드에는 도트(.) 연산자를 사용해 접근함:

```java
// 필드 값 변경
person1.setName("박민수");
person1.setAge(26);

// 필드 값 조회
String name = person1.getName();
int age = person1.getAge();

System.out.println("이름: " + name + ", 나이: " + age);
// 출력: 이름: 박민수, 나이: 26
```

#### 생성자와 초기화 블록

생성자는 객체 생성 시 초기화를 담당함. 초기화 블록은 모든 생성자에서 공통으로 실행될 코드를 포함함:

```java
public class Person {
    private String name;
    private int age;

    // 초기화 블록
    {
        System.out.println("Person 객체 초기화 중...");
    }

    // 생성자
    public Person(String name, int age) {
        this.name = name;
        this.age = age;
        System.out.println("Person 객체 생성 완료!");
    }
}

// 사용 예
Person person = new Person("김철수", 25);
// 출력:
// Person 객체 초기화 중...
// Person 객체 생성 완료!
```

#### 클래스와 객체의 실제 활용

클래스와 객체는 실제 세계의 개념을 모델링하는 데 사용됨. 예를 들어, 도서관 시스템을 구현할 때:

```java
public class Book {
    private String title;
    private String author;
    private boolean borrowed;

    public Book(String title, String author) {
        this.title = title;
        this.author = author;
        this.borrowed = false;
    }

    public void borrow() {
        if (!borrowed) {
            borrowed = true;
            System.out.println(title + " 대출 완료!");
        } else {
            System.out.println(title + "은(는) 이미 대출 중입니다.");
        }
    }

    public void returnBook() {
        if (borrowed) {
            borrowed = false;
            System.out.println(title + " 반납 완료!");
        } else {
            System.out.println(title + "은(는) 이미 반납되었습니다.");
        }
    }

    // Getter와 Setter 생략
}

public class Library {
    private List<Book> books;

    public Library() {
        this.books = new ArrayList<>();
    }

    public void addBook(Book book) {
        books.add(book);
        System.out.println("새 책 추가: " + book.getTitle());
    }

    public void listBooks() {
        System.out.println("도서 목록:");
        for (Book book : books) {
            System.out.println("- " + book.getTitle() + " (저자: " + book.getAuthor() + ")");
        }
    }
}
```

이 클래스들을 사용하는 예시:

```java
public class Main {
    public static void main(String[] args) {
        Library library = new Library();

        Book book1 = new Book("자바의 정석", "남궁성");
        Book book2 = new Book("effective java", "조슈아 블로크");

        library.addBook(book1);
        library.addBook(book2);

        library.listBooks();

        book1.borrow();
        book1.borrow(); // 이미 대출 중인 책
        book1.returnBook();
        book2.borrow();
    }
}
```
